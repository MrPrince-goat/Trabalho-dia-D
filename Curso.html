Curso de Informática para Internet

Este curso abrange os principais conhecimentos e habilidades necessárias para o desenvolvimento e gerenciamento de projetos digitais na web. Você aprenderá a criar desde páginas estáticas até aplicações dinâmicas e a gerenciar a infraestrutura necessária.

Conteúdo Aprendido

No decorrer do curso, abordamos temas essenciais, como:

  • **HTML** (HyperText Markup Language): Estrutura e marcação de conteúdo para a web.
  • **CSS** (Cascading Style Sheets): Estilização visual, layout e design responsivo de páginas web.
  • **Python**: Linguagem de programação versátil, usada para desenvolvimento back-end, automação e análise de dados.
  • **Banco de Dados**: Conceitos e modelagem de dados para armazenamento eficiente.
  • **SQL** (Structured Query Language): Linguagem padrão para gerenciar e manipular dados em bancos de dados relacionais.
  • **ADM** (Administração de Sistemas/Servidores): Fundamentos de gerenciamento de sistemas operacionais e infraestrutura.
  • **Administração Web**: Gerenciamento de websites, servidores web (como Apache ou Nginx) e questões de hospedagem.
